# A Powerful library.
<h1>Built for everybody, to use for advanced functions in only 1 line of code.</h1>

[![forthebadge](https://forthebadge.com/images/badges/made-with-python.svg)](https://forthebadge.com)
<br>
<h3>the reason for making it.</h3>
<ul>
  <li>the library provides ease of use for the developer and allows them to save time & money.</li>
  <li>the library wraps a lot of advanced functions into 1 liner simple to use functions.</li>
  <li>This library is made for everybody.</li>
</ul>  

You can find the library on pypi here: [Pypi Repository](https://pypi.org/project/PowerWrapper/)
<br>
you can download this library by running `pip install PowerWrapper` in respective command line of your operating system.
